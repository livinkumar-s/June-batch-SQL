Joins 
------
>> It is used to combine two or more tables based on a related column.

INNER JOIN 
-----------
>> It retrives only matching records from both table.
Ex: SELECT colDetails FROM tb1 INNER JOIN tb2 ON relatedColumn;

LEFT OUTER JOIN 
------------------
>> It retrves all record from left table.
>> Only matching records from right table.
Ex: SELECT colDetails FROM tb1 LEFT JOIN tb2 ON relatedColumn;

RIGHT OUTER JOIN 
------------------
>> It retrves all record from right table.
>> Only matching records from left table.
Ex: SELECT colDetails FROM tb1 RIGHT JOIN tb2 ON relatedColumn;

FULL OUTER JOIN 
-------------------
>> It retrives all records from both tables.
Left Join (union) right join.

CROSS JOIN 
----------
>> matches every row from left table to every row from right.
Ex: SELECT colDetails FROM tb1 CROSS JOIN tb2;

rows from left * rows right table

SELF JOIN 
---------
>> Table is joined with itself.
